My neovim config, based on [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim)

If you for some reason want to use this

- Backup your current config
```sh
mv ~/.config/nvim ~/.config/nvim-bkp
mv ~/.local/share/nvim ~/.local/share/nvim-bkp
```
- Clone the repo to ~/.config/nvim
```sh
git clone https://github.com/cookie1170/nvim-config ~/.config/nvim
```

- Start neovim
```sh
nvim
```
